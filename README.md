# Kinam

## Start App:
`npm install`
`npm run build:all`
`npm start`

## Routes:
`/`:            Landing
`/playbook`:    Playbook

## Modules:
`core`:         Contains modules needed for the root component (e.g. HttpClientModule)
`shared`:       Contains modules needed for every new module (e.g. FormsModuele)
`nahual-date`:  Contains a class which calculates Nawales
